Visiri - Automated Cell Counter

  • Live
  • Client Repo
  • API Repo

Visiri allows users to upload cell microscopy images (or any image) and create datasets that will allow deep learning models to automatically count cells or classify regions of an image.

In this current build, users can upload images of their cells and label the cells in a mobile-friendly browser-based interface. Intuitive manipulation of image zoom and panning is important to workflow and Visiri supports touch events such as pinch zooming and dragging as well as desktop mouse/wheel events.

The next build will establish the convolutional neural network models and deploying GPU-based architecture for training/prediction and task scheduling.

Created with: React, Javascript, Express, Node.js, Postgres

Bookmark Organizer - Import. Organize. Save.

  • Live
  • Client Repo
  • API Repo

This application parses a user's browser bookmarks and allows them to organize, save, and export back into their browser. The frontend suports drag and dropping, tagging, search/sort/filter, and a persistent storage of their bookmark lists in a normalized relational database.

Created with: React, Express, Node.js, Postgres

Test Project

  • Live

This application is a platform for learning Spanish through the technique of spaced repetition to help with memorization. With this application the user can create an account and login. They can see a dashboard to view their learning progress. The user is able to 'Start Practicing' where they will be given a word that they must type in the translation for. The user will be notified if their answer is correct or incorrect. The backend keeps tracks of scores and decides how to serve new questions based on manipulations to a linked list of questions.

Created with: React, Express, Node.js, Postgres
